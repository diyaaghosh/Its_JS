# JavaScript Mini Projects Website

This repository contains a collection of mini JavaScript projects (beginner level) built to practice and strengthen core web development skills.

## Purpose
After starting my learning journey with Machine Learning, I explored Web Development and found it highly interesting. To improve my understanding of HTML, CSS, and JavaScript, I created several small projects focusing on logic building and DOM manipulation.

## What this project includes
- A single portfolio website showcasing all mini projects
- Individual folders for each project with separate HTML, CSS, and JavaScript files
- Clean and beginner-friendly project implementations
- Live demos and source code access for every project

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

## Goal
The goal of this repository is to demonstrate hands-on learning, improve problem-solving skills, and maintain a structured record of JavaScript practice projects.
